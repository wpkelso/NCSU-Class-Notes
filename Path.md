# Path

A path can be defined as a sequence of edges from $e_{i_{1}}$ to $e_{i_{k}}$.

>[!note] Definition
>**Simple path:** the path does not include the same [[Edge|edge]] twice
>**Elementary path:** the path does not meet the same [[Vertex|vertex]] twice.
